# zipprman
A lightweight application that compresses and decompresses files and directories.
---
## Problem
Simplyfing the binary representation of the file/directory using the Huffman encoding/decoding for easier to transport over internet.
---
## Used technologies
- Language: C++, Interface: Gtk 4.0, CMake for linker
---
## Functionality
- file browser with basic operations: create, delete and move files and directories
- file/directory compression/decompression using the file browser
- selection tab
- optional, multiple compressions happening at the same time using multi-threding
